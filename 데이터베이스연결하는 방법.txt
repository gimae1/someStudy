jsp에서 DB연결하는 방법

Database별
Oracle 10g => 설치한 oracle 폴더 안에 있는 ojdbc14.jar 파일을
WEB-INF폴더 안의 lib폴더 안에 넣는다. 
Oracle 11g =>설치한 oracle 폴더 안에 있는 ojdbc6.jar 파일을
WEB-INF폴더 안의 lib폴더 안에 넣는다. 

MariaDB 10.3 or 10.4=> mvnrepository 사이트에서 mariadb검색 -> maria java client 조회 수 가장 많은 것 찾아서 클릭해서 나온 버전 중에 자신한테 맞는 버전에 맞는 버전 클릭 2.7.3버전 클릭 -> jar파일 다운 => mariadb-java-client.jar ->WEB-INF폴더 안의 lib폴더 안에 넣는다. 

Mysql 5.1.38버전 => mvnrepository 사이트에서 Mysql connect/j검색 -> Mysql connect/j클릭 -> 5.1.38클릭 -> 버전 중에 자신한테 맞는 버전에 맞는 버전 클릭 5.1.38버전 클릭 -> jar파일 다운 =>mysql-connector-j.jarWEB-INF폴더 안의 lib폴더 안에 넣는다. 

db연결할 java파일 만들기 (class만들기)
java.sql패키지에서 제공되는 Connection, PreparedStatement, Result객체 사용하여 DB연결하기
>>가져와야 할 DB 공통 정보 : url, username, password
-데이터베이스별 url정보
oracle =>  "jdbc:oracle:thin:@localhost:1521:xe"
jdbc:oracle:thin:@ => 기본으로 적어야할 값
localhost: => ip주소나 도메인주소
1521: => 오라클 포트번호
xe => 오라클 전역데이터베이스명(=SID)

mariaDB => "jdbc:mariadb://localhost:3306/swork"
mysql => "jdbc:mysql://localhost:3306/boardweb?characterEncoding=utf-8"

username/password(사용자 계정/비번) 자신이 설정한 것으로 설정
----------------------------------------------
>>DB연결할 수 있는 드라이버를 로딩시켜놓는다.(준비시켜놓기)
oracle =>Class.forName("oracle.jdbc.driver.OracleDriver"); 
mariadb =>Class.forName("org.mariadb.jdbc.Driver");
mysql =>Class.forName("com.mysql.jdbc.Driver"); 

--가져올 계정을 설정하여 connection객체를 연결해준다.
--Connection객체 생성
DriverManager.getConnection(url, user, pw);


>>Spring jdbc연결방법
Oracle 10g (ojdbc14)
Oracle 11g (ojdbc6)
MariaDB 10.3 or 10.4 (mariadb-java-client)
Mysql 5.1.38버전 (mysql-connector-j)
mvnrepository => 각 DB별 검색하여 나오는 곳에서 버전 선택후 => maven탭에 나오는 dependency소스코드 복사해서 프로젝트의 pom.xml파일에 <dependencies></dependencies>사이에 붙여넣기. 
되도록 DB연결 정보는 맨 윗줄에 붙여넣기 해준다.
